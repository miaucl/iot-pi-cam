# Summary

Just a quick camera enclosure / stand I whipped up for my Raspberry Pi.

**Be aware:** this enclosure does not fit the v1.3 camera!

# Instructions

Not much assembly required. Just print the 3 parts, grab a 3mm x 25mm screw and nut.  

The tolerance for the lid is .25mm which works good for PLA on my 3D printers. I've added the design files (.f3d or .igs) if you need to make adjustments.

Made in Inventor Fusion for the Mac.
